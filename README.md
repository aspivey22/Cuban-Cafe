# Cuban-Cafe
Cuban Cafe is a Visual Basic Windows Forms application that simulates a simple restaurant ordering system. Users can select food items (such as Cuban sandwiches, chicken dishes, and drinks), and the program automatically calculates individual prices and updates the running total.
# Features
*Item selection withh automatic price calculation

*Live total that updates as items are added

*New Order button to reset totals and start fresh

*Clean, easy-to-use interface

*Beginner-friendly Visual Basic project structure
# Technologies Used
*Visual Basic .NET

*Windows Form (WinForms)

*Visual Studio
# Purpose
This project was created for class to demonstrate basic GUI design, event handling, and real-time price calculations in Visual Basic.
